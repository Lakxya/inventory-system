# 🚀 Inventory System

A simple inventory management system built using HTML, CSS, JavaScript and Supabase.

## 🔥 Features
- Add / Update / Delete products
- User authentication
- Secure data using Row Level Security (RLS)
- Clean UI dashboard

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Supabase
