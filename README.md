# Inventory System

A modern SaaS-style inventory management system with authentication.

- Email & Google Login
- Inventory Management (CRUD)
- Dashboard UI
- Password Reset

## 🛠 Tech Stack

- JavaScript
- Supabase (Auth + DB)
